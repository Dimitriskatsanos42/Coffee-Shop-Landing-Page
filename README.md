# Coffee Shop Landing Page ☕

Μια μοντέρνα και πλήρως ανταποκρίσιμη (responsive) ιστοσελίδα για ένα καφέ-brunch bar στην καρδιά της Θεσσαλονίκης. Σχεδιασμένη με έμφαση στην κομψότητα και την ευκολία χρήσης.

## ✨ Χαρακτηριστικά
* **Responsive Design:** Προσαρμόζεται τέλεια σε κινητά, tablets και υπολογιστές.
* **Interactive Menu:** Δυναμική εναλλαγή ανάμεσα σε κατάλογο φαγητού (Brunch & Pancakes) και ροφημάτων (Coffee & Drinks).
* **Reservation Form:** Ενσωματωμένη φόρμα για κρατήσεις τραπεζιών.
* **Modern UI/UX:** Καθαρό design με χρήση του W3.CSS framework και Google Fonts (Inconsolata).

## 🛠️ Τεχνολογίες
Το project αναπτύχθηκε χρησιμοποιώντας:
* **HTML5** - Δομή περιεχομένου.
* **CSS3 (W3.CSS)** - Styling και layouts.
* **JavaScript** - Λειτουργικότητα των Menu Tabs.
* **Google Fonts** - Τυπογραφία.
