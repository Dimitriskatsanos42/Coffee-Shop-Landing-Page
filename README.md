# ☕ Coffee Shop Landing Page

Μια σύγχρονη, υψηλής αισθητικής και πλήρως ανταποκρίσιμη (responsive) ιστοσελίδα για ένα καφέ-brunch bar στην καρδιά της Θεσσαλονίκης. Το project επικεντρώνεται στην ταχύτητα φόρτωσης και τη βέλτιστη εμπειρία χρήστη (UX).

---

## ✨ Χαρακτηριστικά (Features)

### 📱 Responsive Design
Η σελίδα είναι σχεδιασμένη με την "Mobile First" φιλοσοφία του **W3.CSS framework**, εξασφαλίζοντας ότι η προβολή είναι άψογη σε:
- Smartphones (iOS & Android)
- Tablets
- Desktop Browsers

### 🍴 Δυναμικό Μενού (Interactive Tabbed Menu)
Υλοποίηση με **JavaScript** που επιτρέπει στον χρήστη να περιηγείται μεταξύ των κατηγοριών "Brunch & Pancakes" και "Coffee & Drink" άμεσα, χωρίς ανανέωση της σελίδας, βελτιώνοντας το performance.

### 📅 Σύστημα Κρατήσεων
Ενσωματωμένη φόρμα επικοινωνίας και κρατήσεων με:
- HTML5 Validation για σωστή εισαγωγή στοιχείων.
- Πεδίο `datetime-local` για ακριβή επιλογή ημερομηνίας και ώρας.

### 🎨 Design & Typography
- **Framework:** Χρήση του W3.CSS για ελαφρύ και καθαρό κώδικα χωρίς βαριά αρχεία.
- **Fonts:** Ενσωμάτωση της Google Font `Inconsolata` για μια retro-modern αίσθηση.
- **Visuals:** Grayscale φίλτρα και parallax effects για επαγγελματικό φινίρισμα.

## 🛠️ Τεχνολογική Ανάλυση (Tech Stack)

| Τεχνολογία | Χρήση |
| :--- | :--- |
| **HTML5** | Σημαντική δομή και Semantic Tags (header, footer, section). |
| **CSS3** | Custom styling, Background images και transitions. |
| **W3.CSS** | Framework για το Grid system και το Responsive layout. |
| **JavaScript** | DOM Manipulation για τον έλεγχο προβολής του μενού. |
| **Google Fonts API** | Φιλοξενία γραμματοσειρών για υψηλή αναγνωσιμότητα. |
